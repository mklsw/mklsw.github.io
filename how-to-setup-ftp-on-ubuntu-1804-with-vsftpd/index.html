ubuntu 1804 上配置FTP

Posted on 2018-11-23

趁着黑五在搬瓦工撸了一台VPS，想把自己的一些常用工具放上去，并使用来FTP访问。下面记录一下安装FTP服务的过程。
安装之前我先在自己的虚拟机测试了一遍，没问题后开始安装，过程不是很复杂。

Server: Ubuntu 1804
Tools： Putty, Winscp

  1. 安装vsftpd服务

    sudo apt install vsftpd

  2. 配置vsftpd服务
    先把vsftp服务默认配置文件备份一下

    sudo mv /etc/vsftpd.conf /etc/vsftpd.conf_orig

    再创建一个新的配置文件

    sudo nano  /etc/vsftpd.conf

    再插入以下配置信息：


    接下来就是防火墙配置，没有防火墙的可以不设置

    sudo ufw allow from any to any port 20,21,10000:10100 proto tcp

    启动vsftp服务

    sudo service vsftp restart

  3. 创建FTP的用户名和密码

    sudo useradd -m  mkls
sudo passwd mkls

  4. 测试

    winscp 或直接直接本地测试

    ftp  ftpserveraddress

需要注意的地方：

配置文件不能写错了，修改配置后需要重新启动服务才会生效。
