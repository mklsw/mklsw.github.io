windows 10 1809 安装RSAT

Posted on 2018-11-01

windows 10 升级到1809后去官网下载RSAT时，发现从1809之后，这个工具已经内置在windows 10中，是一个按需安装的功能 即features on demand,不再需要额外的安装包。
关于Features on demand可以参考此链接：featurn on demand

下面是安装的过程：

  1. GUI 安装,比较简单,选自己需要的包就行，看图：

  2. 使用命令行安装，一条命令即可，安装所有的RSAT工具：

    Get-WindowsCapability -Online | Where-Object { $_.Name -like "RSAT*" -and $_.State -eq "NotPresent" } | Add-WindowsCapability -Online

    如果报错了，如下图：

    是因为windows update 环境里面使用WSUS 导致的，需要手动修改一下本地组策略,路径
    "Computer Configuration\Administrative Templates\System\

  3. 如果只需要安装一个包，可以按一下步骤：
    安装之前我们可以查看一下安装包信息：

    DISM /online /get-capabilities /format:table |more

    比较多可以输出到本地查看：

    DISM /online /get-capabilities /format:table >C:\dism.txt

  4. 找到包的名字复制下来，运行一下命令即可：

    DISM.exe /Online /add-capability /CapabilityName:Rsat.ActiveDirectory.DS-LDS.Tools~~~~0.0.1.0

    大功告成
