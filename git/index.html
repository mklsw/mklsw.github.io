Git 新手配置

Posted on 2018-10-27

Git 也是我在学习Python的时候才开始了解的，我理解它就是一款用来控制代码版本的一个工具。下面是初学者的入门配置。

  1. 下载安装

    下载 ：Git for windows 国内下载站

    安装过程省略。

  2. 配置Git
    在任意地方右键点Git bash here 

    $ git config --global user.name "你的名字"

$ git config --global user.email 你的邮件地址

  1. 生成SSH key

    $ssh -keygen
  1. 将生成的SSH Pub Key 添加到Github 或其他地方。
    public key 在windows 10 系统中默认的路径是C:\Users\你的用户名.ssh

Git 入门指引: http://blog.a0z.me/2014/05/21/GitBeginning/
