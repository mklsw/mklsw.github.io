Hyper-V An Error Occurred

Posted on 2018-10-29

连接虚拟机时报错了如下图：

An error occurred while attempting to connect to server “computer name”.Check that the Virtual Machine Management service is running and that you are authorized to connect to the server.
Hyper-V encountered an error trying to access an object on computer “computer name” because the object was not found.The object might have been deleted.Verify that the Virtual Machina Management service on the computer is running.

查了日志没有发现信息。重新卸载安装一遍Hyper-V后自己好了，但在另一台机器上也出现了这个问题。最后在网上找到了解决方案，用一条命令：

MOFCOMP %SYSTEMROOT%\System32\WindowsVirtualization.V2.mof

# Hyper-V # troubleshooting
Windows 10中安装Hyper-V
Windows Server 2016 Server Core 环境下安装DC
